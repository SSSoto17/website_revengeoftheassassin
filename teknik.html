Tehcnique

Get a peek into the inner workings of the code that makes the game function. It runs entirely by javascript, and on this page can be found the UML and SMD diagrams I used to plan my game.

The UML activity diagram shows the general flow of gameplay. It plans out the entire experience from the perspective of the user from beginning to end. I used this as the starting point for development of my game and made adjustments throughout the coding process.

The state machine diagram was created with the process of development in mind. With this diagram, I reflected on what needed to happen at every step of user interaction within the code itself. It helped me plan out functions for my script so the game would work as intended without getting overwhelmed by the details.
